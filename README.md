# Narrify — Dynamic Interactive Story Engine

A simple branching story engine built with Flask, SQLite, and plain HTML/CSS/JS.

## Project Structure

```
narrify/
├── backend/
│   ├── app.py          # Flask API
│   ├── init_db.py      # Creates tables + sample story
│   └── database.db     # Auto-generated by init_db.py
├── frontend/
│   ├── index.html      # Story library (home page)
│   ├── player.html     # Story player
│   ├── editor.html     # Story editor
│   └── style.css       # Shared styles
└── README.md
```

## Setup & Running

### 1. Install dependencies

```bash
pip install flask flask-cors
```

### 2. Initialise the database

```bash
cd backend
python init_db.py
```

This creates `database.db` and inserts a sample story called **"The Forest Path"**.

### 3. Start the backend

```bash
python app.py
```

The Flask server runs at `http://127.0.0.1:5000`

### 4. Open the frontend

Open `frontend/index.html` directly in your browser (double-click or drag into browser).

## How It Works

### Playing a Story
1. Open `index.html` — you'll see the story library
2. Click **Play** on any story
3. Read the scene and click a choice to progress
4. The story branches based on your choices
5. Progress is saved per session (survives page refresh)

### Creating a Story
1. Open `editor.html` or click **Story Editor** in the nav
2. Follow the 4-step process:
   - **Step 1** — Give your story a title and description
   - **Step 2** — Add scenes (narrative text, mark endings)
   - **Step 3** — Link scenes with choices
   - **Step 4** — Set the starting scene and publish
